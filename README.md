## 操作指令
```
npm init
npm i -D rollup typescript
npm i -D @open-wc/building-rollup rimraf deepmerge
```

### 產生 tsconfig.json 範本
```
tsc --init
```

### 轉譯並執行
```
npm run build
npm start
```
